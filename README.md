Scarlet は[『東方Ploject』](https://ja.wikipedia.org/wiki/東方Project)を原作とした二次創作MODです。
紅魔館を中心に、様々な武器やアイテムを追加します。  
詳細については[MOD Docs.md](MOD%20Docs.md)をご覧ください。
