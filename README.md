Scarlet は『[東方Ploject](https://ja.wikipedia.org/wiki/%E6%9D%B1%E6%96%B9Project"東方Ploject")』を原作とした非公認二次創作MODです。
紅魔館を中心に、様々な武器やアイテムを追加します。
